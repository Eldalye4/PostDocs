PostDocs

Free fake REST posts API for testing and prototyping.
Powered by Django + Sqlite .

Resources

PostDocs comes with a set of 3 common resources:

  • /posts - 200 posts
  • /comments - 100 comments
  • /users - 15 users

Note: Resources have relations. For example: users have many posts, ... see guide for the full list.
